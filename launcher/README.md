# ランチャー (Windows用)

Pythonがインストールされていれば、これらの`.bat`ファイルでGUIを起動できます。

## 使い方

### 初回のみ

**`setup_必要なら実行.bat`** をダブルクリック
→ 必要ライブラリ (requests, Pillow, beautifulsoup4) を自動インストール

### 通常の起動

**`ZZZ_Stats起動.bat`** をダブルクリック
→ コンソール非表示でGUIが立ち上がる (`pythonw.exe`経由)

## デスクトップショートカット化

1. `ZZZ_Stats起動.bat` を右クリック → **ショートカットの作成**
2. できたショートカットをデスクトップに移動
3. (任意) 右クリック → プロパティ → アイコンの変更 でお好みのアイコンに

これでデスクトップから1クリック起動できます!
